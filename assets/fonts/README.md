# Custom fonts

Drop `.ttf` / `.otf` files here and Bakuretsu will prefer them over system
fonts. Name bold variants with "bold" in the filename (e.g. `MyFont-Bold.ttf`)
so they are picked for titles and scores.

Note: if you review in Arabic, make sure the font supports Arabic script —
the bundled fallbacks (Segoe UI / Tahoma / Arial) already do.
